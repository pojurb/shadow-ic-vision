# Vision Fixture Provenance

These PNG files are Playwright-rendered HTML pages, not real company filings
or scanned documents. They exist to give the M005 OCR/vision eligibility eval
genuine image bytes to transcribe (rather than a JSON description of a
document), without using an actual confidential or copyrighted filing.

Generated by `scripts/generate-vision-fixtures.ts` (`npm run fixtures:vision:generate`).

| File | Candidate quote a passing vision model must transcribe |
|---|---|
| `pltr-gross-margin-scan.png` | Gross margin for the quarter was 81.3%, compared to 78.1% in the prior-year period. |
| `bbri-nim-scan.png` | Pendapatan bersih meningkat 12,4% dibandingkan periode yang sama tahun lalu. |
